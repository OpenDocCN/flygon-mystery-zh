# 十五、爱情法术

巫术和施法是分不开的。正如你已经知道的，巫术是使用你的意志、头脑、情感，或许还有特殊工具来实现你的意图的实践——魔法就是在这个过程中发生的事情。魔法不一定要复杂才能成功。如果有的话，越简单越好——除非大量的细节和戏剧能让你创造出你想要的魔法氛围。有些咒语只需要女巫在场就能激活。

在神话、歌曲和诗歌中，爱情和魔法经常联系在一起。当我们谈到坠入爱河时，我们会使用诸如被施了魔法、被迷住了、被迷住了、被情人施了魔法之类的术语——而且我们经常会有这种感觉。但是正如莎士比亚在他那部关于魔法和爱情的令人愉快的戏剧《仲夏夜之梦》中所写的，“真爱的道路从来都不是平坦的。”因此，最常见的咒语是——你猜对了——爱情咒语也就不足为奇了。

然而，爱情咒语是很复杂的，不少女巫因为浪漫的原因施过可疑的咒语。很容易让你的心控制你的头脑。在你使用本章中的咒语之前，想一想你在一段关系中到底想要什么，以及你施咒的动机是什么。你真的想用魔法抓住一个不那么喜欢你的男人吗？如果你女朋友出轨了，你真的想让她回来吗？

## 定义你想要什么

你施爱的咒语的原因是什么？你想吸引一个新的浪漫伴侣吗？寻找你的灵魂伴侣？希望重新点燃现有关系的火花？你越具体，你成功的机会就越大。

请记住，爱情魔咒的主要目的不是用魔法来操纵某人爱上你。它的目的是平衡你自己的能量，这样你就能吸引到适合你的伴侣。一段关系是两个人相互作用的结果，两种独立的力量——阴和阳——合在一起形成一个大于部分的总和。

你可能认为这应该很容易，但对许多人来说并不容易。你的想法可能会受到你的家人、朋友或你所生活的文化的影响。你对理想伴侣的看法可能会受到电视和电影、书籍和杂志以及各种社会刻板印象的影响。津巴布韦男人结婚的原因可能与瑞典夫妇结婚的原因大不相同。甚至你和你最好的朋友可能也不同意什么是好的关系。

### 你的爱情清单

在你开始任何爱情咒语之前，列出一份你在伴侣身上寻求的品质清单。编制这个清单的行为会促使你真正思考你的需求、欲望、希望和优先事项，并在创造你想要的东西的过程中投入精力和意图。从某种意义上说，你的清单变成了一个咒语。像肯定一样，你陈述你的意图——你打算在伴侣身上找到什么——并集中精力实现你的目标。

你的清单可以有多长有多详细，由你选择。当你写清单时，一定要用积极的方式和现在时态来陈述你的愿望。以下是一些例子:

*   我现在有了一个尊重和重视我的伴侣。
*   在任何时候、任何情况下，我都可以信任和依赖我的伴侣。
*   我和我的伴侣支持和鼓励彼此的目标。
*   我和我的伴侣有着共同的精神道路。
*   我和我的伴侣有许多共同的兴趣，喜欢彼此的陪伴。
*   我现在有了一个伴侣，他愿意并且能够和我建立一种忠诚的、爱的、主要的伙伴关系。

具体点。考虑每一个角度。覆盖你所有的基地。你可能想让一个信任的朋友阅读你的清单并提供反馈，这样你就不会忽略任何东西。你不想写一份长长的详细的爱情清单，描述你在伴侣身上寻求的所有品质，然后意识到你遗漏了一些重要的东西时已经太晚了。我曾经列出了我希望伴侣具备的条件，其中一项是“喜欢旅行”我应该说“喜欢和我一起旅行”*，因为我吸引的那个男人喜欢旅行——而且确实很喜欢——他只是更喜欢独自旅行。*

在护身符上洒上盐水，然后将它放在熏香的烟雾中一会儿，给它充电。打开圆圈。一直戴着或带着护身符，直到那个讨厌的人不再打扰你。

## 改编咒语

有时候，你可能想要修改一个法术来满足你的特殊需要，但是你从哪里开始这个过程呢？虽然改编一个咒语远比创造一个容易，但这仍然需要一些深谋远虑。在检查咒语时，要注意连续性和全面性。问自己这些问题:

1.  这个咒语真的用它的语言、动作和成分来瞄准你的目标吗？
2.  它是否在多种感官层面上做到了这一点，调动了你的听觉、视觉、触觉、味觉和嗅觉？
3.  咒语的每一部分都给你灵感了吗？

如果以上任何一个问题的答案是否定的，你应该试着找一个替代品。例如，许多古老的爱情咒语需要血液作为成分。然而，有许多健康原因说明这在今天可能不是一个好主意——此外，它可能会让你感到恶心。相反，你可以用红色果汁或红酒来代替——你的意图是你的咒语成功的最重要的组成部分。如果你不喜欢某个咒语中的一些词，编辑这首诗或圣歌让它对你说话没什么不对。很多人对某些物质过敏，或者只是不喜欢精油或熏香的味道——所以继续使用你喜欢的东西。

在许多情况下，和你的爱人一起施爱情魔咒是很棒的(假设他或她愿意并同意你)。可以说，你甚至可以将性魔法融入到你的拼写中来增加辛烷值——但是在开始之前要确保你知道你在做什么(我的书*初学者的性魔法*解释了你需要知道什么来开始)。*
